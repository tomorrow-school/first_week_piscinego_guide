## Разбор задания: IsNegative

* Функция: func IsNegative(nb int)
* Тип данных: Функция принимает одно целое число (int).
* Логика:
    * Если число отрицательное ($nb < 0$), вывести T.
    * Если число положительное ($nb > 0$) или равно нулю ($nb = 0$), вывести F.
* Инструмент: Для печати, как и в других заданиях Piscine, используется функция z01.PrintRune.
* Формат вывода: Результат (T или F) должен быть выведен в отдельной строке. Это означает, что после T или F необходимо напечатать символ новой строки (\n).
* Вывод:
    * T
    * F
    * F
    * $

---

## Решение

```go
package piscine

func IsNegative(nb int) {
    if nb < 0 {
        z01.PrintRune('T')
    } else {
        z01.PrintRune('F')
    }
    z01.PrintRune('\n')
}

```

## Разбор логики решения

Задание является простым условным оператором (if-else) с обязательным требованием печати символа новой строки (\n) после результата.
Основные шаги алгоритма:

Проверка условия: Используется оператор if для проверки, является ли входной параметр nb меньше нуля (nb < 0).

Печать "True": Если условие истинно, печатается руна 'T', за которой следует руна новой строки ('\n').

Печать "False": Если условие ложно (ветвь else), печатается руна 'F', за которой следует руна новой строки ('\n').

## Заключение
Задание IsNegative — это простейшая задача на условное ветвление (if-else).
1. Основная задача: Свести любое входное целое число (int) к бинарному результату: отрицательное (T) или не отрицательное (F).
2. Ключевой момент: Обязательно нужно обеспечить перевод строки (z01.PrintRune('\n')) после символа T или F для соответствия формату вывода задания. Ноль (0) считается не отрицательным, и для него выводится F.

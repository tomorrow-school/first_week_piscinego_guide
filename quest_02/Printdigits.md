## Разбор задания: printdigits

* Название файла: printdigits/main.go
* Задача (Instruction): Написать программу, которая печатает десятичные цифры в возрастающем порядке (от 0 до 9) на одной строке, за которой следует символ новой строки (\n).
* Разрешенные функции: github.com/01-edu/z01.PrintRune, а также встроенные (built-in) функции Go, такие как циклы и арифметические операции.
* Пример:
    * Запуск: $ go run .
    * Выход: 0123456789

---

## Решение

```go
package main

import "github.com/01-edu/z01"

func main() {
	for r := '0'; r <= '9'; r++ {
		z01.PrintRune(r)
	}
	z01.PrintRune('\n')
}
```

## Ключевые понятия

| **Инструмент / элемент кода** | **Объяснение и назначение** |
|-------------------------------|------------------------------|
| **package main** | Определяет, что это исполняемый пакет (главная программа) |
| **import "github.com/01-edu/z01"** | Импортирует внешний пакет z01 для использования функции PrintRune. |
| **func main() {** | Главная функция, с которой начинается выполнение программы Go. |
| **for r := '0'; r <= '9'; r++ {** | Цикл for для итерации по символам-цифрам: — Начало: r := '0' объявляет руну r и присваивает ей код символа '0'. — Условие: Цикл продолжается, пока код руны меньше или равен коду символа '9'. — Шаг: r++(инкремент) увеличивает код руны на 1, переходя к следующей цифре (например, от '1' к '2'). |
| **z01.PrintRune(r)** | Выводит текущую руну (r) на стандартный вывод, печатая цифру. |
| **z01.PrintRune('\n')** | Выводит символ новой строки (\n), который требуется для завершения вывода на одной строке. |

## Глоссарий

| **Термин** | **Понятное объяснение** |
|-------------|--------------------------|
| **Руна (Rune)** | В Go это псевдоним для типа int32, используемый для представления символов Unicode. В отличие от байта, руна корректно обрабатывает символы любого языка. |
| **Символ в кавычках ('0')** | В Go символ, заключенный в одинарные кавычки, является руной и представляет собой целое число (код символа). r := '0' эквивалентно r := 48 (ASCII-код). |
| **Инкремент (r++)** | Оператор, который увеличивает значение переменной на единицу. В данном случае, увеличивает код руны, переходя к следующему символу в таблице. |
| **package main** | Особый пакет, который определяет, что программа может быть скомпилирована и запущена как самостоятельное приложение. |
| **PrintRune** | Разрешенная функция вывода, которая принимает одну руну и печатает ее. Используется для посимвольного вывода. |

## Название файла Go
`PrintComb2`

---

## Условия задания 

| **Instruction** | **Перевод** |
|---------------------------|--------------|
| *Write a function that prints in ascending order and on a single line: all possible combinations of two different two-digit numbers. These combinations are separated by a comma and a space. * | *Напишите функцию, которая выводит, в порядке возрастания и на одной строке: все возможные комбинации из двух различных двузначных чисел. Эти комбинации должны быть разделены запятой и пробелом* |

## Разбор задания: PrintComb2

* Функция: func PrintComb2()
* Условие комбинации:
    * Комбинация состоит из двух двузначных чисел (от 00 до 99).
    * Числа должны быть различными (например, 00 01 подходит, 00 00 — нет).
* Формат вывода:
    * Вся последовательность должна быть на одной строке.
    * Комбинации должны быть отсортированы по возрастанию (лексикографически).
    * Разделитель между комбинациями: запятая и пробел (, ).
    * Внутри каждой комбинации числа разделены пробелом.
    * Числа должны быть выведены с ведущим нулём при необходимости (например, 00, 01).
* Диапазон: Комбинации от 00 01 до 98 99.
* Ожидаемый неполный вывод (Example): 00 01, 00 02, 00 03, ... 00 98, 00 99, 01 02, 01 03, ... 97 98, 97 99, 98 99

---

## Решение

```go
package piscine

import "github.com/01-edu/z01"

func printN(n int) {
	z01.PrintRune(rune(n/10) + '0')
	z01.PrintRune(rune(n%10) + '0')
}

func PrintComb2() {
	for i := 0; i <= 98; i++ {
		for j := i + 1; j <= 99; j++ {

			printN(i)

			z01.PrintRune(' ')

			printN(j)

			if i != 98 {
				z01.PrintRune(',')
				z01.PrintRune(' ')
			}
		}
	}
}

```

## Заключение
Функция PrintComb2 решает задачу генерации и печати всех уникальных, упорядоченных пар двузначных чисел.

1. Генерация: Используется два вложенных цикла, где внутренний цикл (j) начинается с i + 1. Это позволяет автоматически исключить пары одинаковых чисел (i = j) и гарантирует, что Число1​<Число2​.
2. Форматирование: Используется вспомогательная функция printN и арифметика (/10 и %10) для корректного вывода каждого числа в двузначном формате с ведущими нулями (00 вместо 0).
3. Разделитель: Условный оператор if i != 98 гарантирует, что разделитель , печатается междукомбинациями, но не после последней (98 99), соблюдая строгое требование вывода.

Понимание вложенных циклов, работы с числовыми операциями (/, %) и точное управление выводом с помощью z01.PrintRune являются ключевыми навыками, отрабатываемыми в этом квесте.

# Quest 2. PrintComb

## Название файла Go
`printcomb.go`

---

## Условия задания 

| **Instruction** | **Перевод** |
|---------------------------|--------------|
| *Write a function that prints, in ascending order and on a single line: all unique combinations of three different digits so that the first digit is lower than the second, and the second is lower than the third. These combinations are separated by a comma and a space.* | *Напишите функцию, которая выводит, в порядке возрастания и на одной строке: все уникальные комбинации из трёх разных цифр, при условии, что первая цифра меньше второй, а вторая меньше третьей. Комбинации должны быть разделены запятой и пробелом.* |

---

## Разбор задания: PrintComb

* Функция: func PrintComb()
* Условия комбинации:
    * Состоит из трёх разных цифр (например, 012 подходит, 001 или 011 — нет).
    * Цифры должны быть в строго возрастающем порядке (цифра1​<цифра2​<цифра3​).
* Формат вывода:
    * Все комбинации должны быть на одной строке.
    * Комбинации должны быть отсортированы по возрастанию (лексикографически, т.е. 012 перед 013).
    * Разделитель между комбинациями: запятая и пробел (, ).
* Невалидные примеры:
    * 000 или 999 — невалидные комбинации, потому что цифры не разные.
    * 987 — не должен быть показан, потому что первая цифра не меньше второй (не соблюдено строгое возрастание).
* Ожидаемый неполный вывод (Example): 012, 013, 014, 015, 016, 017, 018, 019, 023, ... 689, 789

---

## Решение

```go
package piscine

import "github.com/01-edu/z01"

func PrintComb() {
    for i := 0; i <= 7; i++ {
        for j := i + 1; j <= 8; j++ {
            for k := j + 1; k <= 9; k++ {

                z01.PrintRune(rune(i + '0'))
                z01.PrintRune(rune(j + '0'))
                z01.PrintRune(rune(k + '0'))

                if i != 7 {
                    z01.PrintRune(',')
                    z01.PrintRune(' ')
                }
            }
        }
    }
}

```

## Ключевые понятия

| **Инструмент / элемент кода** | **Объяснение и назначение** |
|-------------------------------|------------------------------|
| **func PrintComb() {** | Объявление функции. Функция не принимает никаких аргументов, так как генерирует комбинации самостоятельно. Она ничего не возвращает (void функция), а печатает результат напрямую в консоль. |
| **for i := 0; i <= 7; i++ {** | Внешний цикл. Перебирает первую цифру (i) комбинации. Начинается с 0. Ограничение ≤7 гарантирует, что останется место для двух больших цифр (максимальная комбинация: 7, 8, 9). |
| **for j := i + 1; j <= 8; j++ {** | Средний цикл. Перебирает вторую цифру (j). Начинается с i + 1, чтобы гарантировать, что j>i (вторая цифра больше первой) и они разные. Ограничение ≤8 гарантирует место для третьей цифры (максимальное j=8 в комбинации 7, 8, 9). |
| **for k := j + 1; k <= 9; k++ {** | Внутренний цикл. Перебирает третью цифру (k). Начинается с j + 1, чтобы гарантировать, что k>j (третья цифра больше второй) и они разные. Ограничение ≤9 является максимальной цифрой. |
| **z01.PrintRune(rune(i + '0'))** | Печать первой цифры. Преобразует числовое значение i в его символ (руну) для печати. Это делается путем добавления числового значения i к руне '0' (например, 1+’0’=’1’). Аналогичные конструкции используются для печати j и k. |
| **if i != 7 { ... }** | Условие разделителя. Проверяет, является ли текущая комбинация последней (789). Если i не равно 7, значит, это не последняя комбинация, и нужно напечатать разделитель. |
  **z01.PrintRune(',') и z01.PrintRune(' ')** | Печать разделителя. Если условие if выполнилось, печатаются руны запятой и пробела для разделения комбинаций. |
  **Фигурные скобки { }** | Определяют границы блоков кода: тела функции и каждого из трёх вложенных циклов. |
  **Логика циклов i + 1, j + 1** | Ключевой элемент требования. Обеспечивает выполнение двух главных условий задания: 1) Цифры разные; 2) Строго возрастающий порядок (цифра1​<цифра2​<цифра3​). |

## Глоссарий

| **Термин** | **Понятное объяснение** |
|-------------|--------------------------|
| **Вложенный цикл** | Цикл, который находится внутри другого цикла. Используется для перебора комбинаций, где каждая внешняя итерация запускает полный цикл внутренней. |
| **Ascending order** | Порядок возрастания. В этом задании означает, что цифра1​<цифра2​<цифра3​. |
| **Уникальная комбинация** | Группа из трёх цифр, где каждая цифра отличается от других (например, 123 и 321 считаются разными перестановками, но в этом задании, благодаря правилу строгого возрастания, 123 будет единственной валидной комбинацией этих цифр). |
| **func PrintComb()** | Объявление функции. Ключевое слово func (Function), имя PrintComb (с заглавной буквы, что делает её доступной извне), не принимает параметров и не возвращает значения. |
| **Лексикографический порядок** | Порядок, используемый в словарях: 012 идет раньше 013, потому что 2 меньше 3 в первой отличающейся позиции. |
| **Разделитель (, )** | Символы, вставленные между результатами для их отделения друг от друга, но не после последнего элемента. |
| **z01.PrintRune** | Функция, которую обычно используют в "Piscine" для вывода одиночного символа (руны) в консоль. Цифры 0−9нужно предварительно преобразовать в руны, добавив код руны '0'. |

## Заключение
Задание PrintComb — это фундаментальное упражнение, которое учит:

1. Генерации комбинаций: Использовать вложенные циклы с правильными граничными условиями (i≤7, j≤8) и смещением (i+1, j+1) для создания строго возрастающих и уникальных последовательностей.
2. Точному выводу: Преобразовывать числовые значения в символы (+′0′) и управлять печатью разделителейтаким образом, чтобы они появлялись между элементами, а не после последнего.

Успешное выполнение этой задачи демонстрирует крепкое понимание базового управления потоком и работы с символами в Go.

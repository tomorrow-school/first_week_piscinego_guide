# Quest 3. ultimatepointone

## Название файла Go
`ultimatepointone.go`

---

## Условия задания 

| **Instruction** | **Перевод** |
|---------------------------|--------------|
| *Write a function that takes a pointer to a pointer to a pointer to an int as argument and gives to this int the value of 1.* | *Напиши функцию, которая принимает указатель на указатель на указатель на int и присваивает этому числу значение 1.* |

---
 
## Решение

```go
package piscinego

func UltimatePointOne(n ***int) {
    ***n = 1
}
```
---

## Ключевые понятия

| **Инструмент / элемент кода** | **Объяснение и назначение** |
|-------------------------------|------------------------------|
| **package piscinego** | Файл принадлежит пакету `piscinego`, чтобы Go понимал, где лежит функция. |
| **func UltimatePointOne(n \*\*\*int)** | Объявление функции. Принимает тройной указатель на int. |
| **n \*\*\*int** | Тип параметра: указатель на указатель на указатель. Хранит цепочку ссылок. |
| **n** | Переменная внутри функции. Хранит адрес переменной, в которой лежит адрес другой переменной. |
| **\*n** | Разыменование 1 уровня → получаем `**int` (указатель на указатель). |
| **\*\*n** | Разыменование 2 уровня → получаем `*int` (указатель на число). |
| **\*\*\*n** | Разыменование 3 уровня → добираемся до реального `int`. |
| **\*\*\*n = 1** | Присваиваем значение 1 исходной переменной `a` в main. |

---

## Логика и пошаговый разбор кода

### 1. Создаются переменные в main

```go
a := 0
b := &a
n := &b
```
a — обычное число
b — указатель на a
n — указатель на b
Цепочка: n → b → a → 0

### 2. Вызов функции
```go
UltimatePointOne(&n)
```
*Передаём адрес переменной n, тип становится ***int.*

### 3. Интерпретация параметра
*Внутри функции:*
```go
func UltimatePointOne(n ***int)
```
*n — это указатель на указатель на указатель.*
### 4. Разыменование
```go
***n = 1
```
*Раскрываем уровни:*
| Уровень | Значение    |
| ------- | ----------- |
| *n      | указатель b |
| **n     | указатель a |
| ***n    | значение a  |

---

## Глоссарий

| Термин    | Пояснение                        |
| --------- | -------------------------------- |
| Указатель | Хранит адрес переменной          |
| &a        | Взять адрес переменной           |
| *p        | Разыменовать → получить значение |
| **p       | Разыменовать дважды              |
| ***p      | Разыменовать трижды              |
| n ***int  | Тройной указатель                |

---

## Заключение
1. Функция работает с тройным указателем.
2. ***n позволяет добраться до переменной a.
3. Значение a меняется на 1 в main.
4. Упражнение показывает работу с памятью и глубокими указателями.

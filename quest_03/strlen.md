# Quest 3 — StrLen

## Название файла Go
`strlen.go`

---

## Условия задания

| **Instruction** | **Перевод** |
|------------------|-------------|
| *Write a function that counts the runes of a string and that returns that count.* | *Напишите функцию, которая считает количество рун (символов) в строке и возвращает это число.* |

---

## Решение

```go
package piscine

func StrLen(s string) int {
	count := 0
	for range s {
		count++
	}
	return count
}
```

---

## Ключевые понятия

| **Термин / инструмент** | **Объяснение** |
|--------------------------|----------------|
| **Rune (`rune`)** | Unicode-символ (тип `int32`). Один символ может занимать 1–4 байта. |
| **Строка (`string`)** | Последовательность байтов в UTF-8. Строки неизменяемые (immutable). |
| **UTF-8** | Кодировка, в которой один символ может быть многобайтовым. Поэтому длина строки в байтах ≠ количеству символов. |
| **for range по строке** | Перебирает строку по *рунам*, а не байтам. Именно поэтому используется здесь. |
| **Счётчик длины строки** | В Go нельзя просто взять `len(s)` для получения количества символов — это вернёт количество байтов. Правильнее считать через `for range` если там есть не только ASCII символы. |

---

## Логика и пошаговый разбор кода

### 1. Инициализация счётчика

```go
package piscine

func StrLen(s string) int {
    count := 0
```
- Создаём переменную, в которую будем складывать количество найденных символов. Начальное значение `0`.

---

### 2. Перебор строки по символам (рунам)

```go
    for range s {
	    count++
    }
```

- Цикл `for range s` автоматически извлекает одну руну за итерацию.
- Мы не используем индекс и значение (`_ , r`) — нам важно только количество символов.
- Каждый шаг цикла означает один найденный символ, поэтому просто увеличиваем `count++`.

**Почему не `len(s)`?**  
Потому что `len(s)` возвращает **количество байтов**, а не символов.

Пример:  
```
"Привет" → 12 байтов, но 6 символов  
"猫" → 3 байта, но 1 символ  
```
---

### 3. Возврат результата

```go
    return count
```
Функция возвращает количество рун - настоящую длину строки в символах.

---

## Примеры использования

```go
package main

import (
	"fmt"
	"piscine"
)

func main() {
	fmt.Println(piscine.StrLen("Hello"))     // 5
	fmt.Println(piscine.StrLen("Привет"))    // 6
	fmt.Println(piscine.StrLen("猫"))         // 1
	fmt.Println(piscine.StrLen(""))          // 0
}
```

---

## Визуализация работы

Строка: `"Go!"`

### Цикл `for range` итерируется так:

| Итерация | Символ (`rune`) | `count` |
|----------|------------------|---------|
| 1        | `'G'`           | 1       |
| 2        | `'o'`           | 2       |
| 3        | `'!'`           | 3       |

Результат: `3`.

---

## Заключение

Функция `StrLen` показывает важную особенность Go:
> длина строки в байтах **не равна** длине строки в символах.

Использование цикла `for range` гарантирует корректный подсчёт символов в UTF-8, включая многобайтовые руны.  
Этот подход — стандартный и идиоматичный способ считать количество символов в строке на Go.

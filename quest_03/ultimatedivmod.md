# Quest 3. ultimatedivmod

## **Название файла Go**  
`ultimatedivmod.go`
---
## **Условия задания**

| **Instruction** | **Перевод** |
|-----------------|-------------|
| *Create the following function.* | *Создай следующую функцию.* |
| *UltimateDivMod should divide the dereferenced value of a by the dereferenced value of b.* | *Функция должна разделить значение по адресу `a` на значение по адресу `b`.* |
| *Store the result in the int which a points to.* | *Результат деления нужно сохранить по адресу `a`.* |
| *Store the remainder in the int which b points to.* | *Остаток от деления нужно сохранить по адресу `b`.* |
---
## **Решение**

```go
package piscinego

func UltimateDivMod(a *int, b *int) {
    div := *a / *b
    mod := *a % *b
    *a = div
    *b = mod
}
```
---
## Ключевые понятия

| **Инструмент / элемент кода**           | **Объяснение и назначение**                         |
| --------------------------------------- | --------------------------------------------------- |
| **package piscinego**                   | Файл принадлежит пакету `piscinego`.                |
| **func UltimateDivMod(a *int, b *int)** | Функция принимает два указателя на int.             |
| **a *int / b *int**                     | Хранят адреса переменных, а не сами числа.          |
| ***a, *b**                              | Разыменование: получить значения по адресам.        |
| **div := *a / *b**                      | Деление значений, на которые указывают `a` и `b`.   |
| **mod := *a % *b**                      | Остаток от деления значений.                        |
| ***a = div**                            | Записываем результат деления обратно по адресу `a`. |
| ***b = mod**                            | Записываем остаток деления по адресу `b`.           |

## Логика и пошаговый разбор кода

### 1. Параметры функции

```go
func UltimateDivMod(a *int, b *int)
```
a — указатель на число
b — указатель на число
через *a и *b мы работаем с их настоящими значениями

### 2. Считывание значений по указателям
```go
div := *a / *b
mod := *a % *b
```
*a — значение, на которое указывает a
*b — значение, на которое указывает b

Функция вычисляет:
div → результат деления
mod → остаток от деления

### 3. Запись значений обратно
```go
*a = div
*b = mod
```
Здесь мы изменяем исходные переменные, на которые указывали a и b.

То есть:
по адресу a теперь хранится результат деления
по адресу b теперь хранится остаток

## Глоссарий
| **Термин**                      | **Пояснение**                                         |
| ------------------------------- | ----------------------------------------------------- |
| **Указатель (`*int`)**          | Хранит адрес переменной.                              |
| **Разыменование (`*a`)**        | Получить значение переменной по адресу.               |
| **&x**                          | Взять адрес переменной `x`.                           |
| **a / b**                       | Целочисленное деление.                                |
| **a % b**                       | Остаток от деления.                                   |
| **Модификация через указатель** | Позволяет менять оригинальную переменную, а не копию. |

## Пример использования функции
```go
package main

import (
    "fmt"
    "piscinego"
)

func main() {
    a := 10
    b := 3
    var div int
    var mod int

    piscinego.DivMod(a, b, &div, &mod)

    fmt.Println("div =", div) // результат деления
    fmt.Println("mod =", mod) // остаток
}
```
## Ожидаемый вывод
```go
div = 3
mod = 1
```

## Объяснение
10 / 3 = 3 → сохраняется в div
10 % 3 = 1 → сохраняется в mod
Функция работает корректно и возвращает два результата через указатели.

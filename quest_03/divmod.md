# Quest 3. divmod

## Название файла Go
`divmod.go`

---

## Условия задания 

| **Instruction**                                                            | **Перевод**                                                                      |
| -------------------------------------------------------------------------- | -------------------------------------------------------------------------------- |
| *Write a function that will be formatted as below.*                        | *Напиши функцию в формате ниже.*                                                 |
| *This function will divide the int a and b.*                               | *Эта функция должна разделить число `a` на число `b`.*                           |
| *The result of this division will be stored in the int pointed by div.*    | *Результат деления (`a / b`) должен быть записан в переменную по адресу `div`.*  |
| *The remainder of this division will be stored in the int pointed by mod.* | *Остаток от деления (`a % b`) должен быть записан в переменную по адресу `mod`.* |

## Решение

```go
package piscinego

func DivMod(a int, b int, div *int, mod *int) {
	*div = a / b
	*mod = a % b
}
```

## Логика и пошаговый разбор кода

### 1. Объявление функции

```go
func DivMod(a int, b int, div *int, mod *int)
```
Функция принимает:
a, b — обычные числа
div — указатель, куда сохранить результат деления
mod — указатель, куда сохранить остаток

### 2. Запись результата деления
```go
*div = a / b
```
Объяснение:
*div — разыменовываем указатель
по адресу div записывается a / b

Пример:
если a = 10, b = 3, то *div = 3.

### 3. Запись остатка
```go
*mod = a % b
```
Объяснение:
*mod — разыменуем указатель
по адресу mod сохраняем остаток от деления

Пример:
если a = 10, b = 3, то *mod = 1.

### 4. Результат
Функция ничего не возвращает (void),
но меняет значения, лежащие по указателям div и mod.

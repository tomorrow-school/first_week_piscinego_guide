# Quest 3.7 — Swap

## Название файла Go
`swap.go`

---

## Условия задания

| **Instruction** | **Перевод** |
|-----------------|-------------|
| *Write a function that takes two pointers to an `int` (`*int`) and swaps their contents.* | *Напишите функцию, которая принимает два указателя на `int` (`*int`) и меняет их значения местами.* |

---

## Пример использования

```go
package main

import (
	"fmt"
	"piscine"
)

func main() {
	a := 0
	b := 1
	piscine.Swap(&a, &b)
	fmt.Println(a) // 1
	fmt.Println(b) // 0
}
```

**Ожидаемый вывод:**
```
1
0
```

---

## Решение

```go
package piscine

func Swap(a *int, b *int) {
	// Сохраняем значение, на которое указывает a
	temp := *a
	// Записываем в a значение из b
	*a = *b
	// Возвращаем в b значение из temp
	*b = temp
}
```

---

## Ключевые понятия 

| **Термин / инструмент** | **Объяснение** |
|-------------|----------------|
| **Указатель (`*int`)** | Хранит адрес переменной, а не ее значение. Позволяет изменять данные не обновляя переменную а по адресу. |
| **Оператор `*` (разыменование)** | Доступ к значению, на которое указывает указатель. Например: `*a` — значение переменной, хранящейся по адресу `a`. |
| **Оператор `&`** | Получение адреса переменной. Пример: `&a` — указатель на переменную `a`. |
| **swap (перестановка)** | Один из самых распространённых паттернов: временная переменная → запись → возврат значения. |

---

## Что такое указатели/поинтеры (Pointers)?

- **Переменная** хранит **значение**  
- **Указатель** хранит **адрес**, где лежит значение  

Пример:

```go
x := 10
p := &x // p хранит адрес переменной x
```

### 1. Разыменование (`*p`) даёт доступ к значению по адресу

```go
*p = 99 // теперь x == 99
```

### 2. Почему swap нельзя сделать без указателей?

Потому что Go передаёт аргументы **по значению**, а не по ссылке:

```go
func fail(a, b int) {
    temp := a
    a = b
    b = temp
}
```

Этот код **не изменит исходные переменные**, потому что `a` и `b` — копии.

## Логика и пошаговый разбор кода

### 1. Объявление функции

```go
package piscine

func Swap(a *int, b *int) {
```

- принимаем `a` и `b` — **указатели на int**
- мы будем менять **значения по адресам**, а не копии

---

### 2. Создание временной переменной

```go
    temp := *a
```

- `*a` — значение по адресу `a`
- сохраняем его, чтобы не потерять при перезаписи

---

### 3. Сохраняем значение b в a

```go
    *a = *b
```

Теперь в `a` лежит значение, на которое указывал `b`.

---

### 4. Возвращаем temp в b

```go
    *b = temp
}
```
Перестановка завершена.

## Заключение

Функция **Swap** — базовый пример работы с указателями:

- мы изменяем данные **по адресу**, а не переменные внутри функции  
- используем временную переменную для перестановки  
- понимаем разницу между значением переменной и адресом

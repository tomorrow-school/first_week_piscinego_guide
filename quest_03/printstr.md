# Quest 3.5 PrintStr

## Название файла Go
`printstr.go`

---

## Условия задания

| **Instruction** | **Перевод** |
|------------------|-------------|
| *Write a function that prints one by one the characters of a string on the screen.* | *Напишите функцию, которая выводит на экран символы строки по одному.* |

---
## Пример использования

```go
package main

import (
	"piscine"
)

func main() {
	piscine.PrintStr("Hello")
	piscine.PrintStr("12345")
	piscine.PrintStr("Go Piscine!")
}
```

Вывод:
```
Hello12345Go Piscine!
```
---

## Решение

```go
package piscine

import "github.com/01-edu/z01"

func PrintStr(s string) {
	for _, r := range s {
		z01.PrintRune(r)
	}
}
```

---

## Ключевые понятия

| **Термин / инструмент** | **Объяснение** |
|--------------------------|----------------|
| **Тип `string`** | Последовательность символов в кодировке UTF-8. Строки в Go неизменяемые (immutable), поэтому операции над строками создают новые значения. |
| **Тип `rune` (`int32`)** | Числовое значение Unicode-символа. Каждая запись `for range` по строке возвращает именно `rune`. |
| **UTF-8** | Стандартная кодировка строк в Go. Один символ может занимать 1–4 байта. Именно поэтому безопаснее работать с *рунами*, а не с байтами. |
| **for range по строке** | Перебирает строку по Unicode-символам. Каждый символ приходит как `rune`, даже если он многобайтовый. |
| **z01.PrintRune** | Специальная функция для вывода **одного символа (руны)**. Это требование заданий Piscine использовать именно ее вместо `fmt`. |

---
## Логика и пошаговый разбор кода

### 1. Объявление функции

```go
func PrintStr(s string) {
```
- Функция принимает **строку** `s`.
- Ничего не возвращает, ее задача просто *вывести* символы строки на экран.
---

### 2. Перебор всех символов строки

```go
    for _, r := range s {
```
- Цикл `for range` это самый удобный способ пройтись по строке.  
- Он автоматически разбивает строку на **руны**, а не байты.
- `_` — placeholder для индекса символа (который нам не нужен),  
  `r` — значение символа в виде `rune`.

Это важно, так как строка может содержать не только ASCII, но и Unicode-символы.

---

### 3. Вывод каждого символа по одному

```go
        z01.PrintRune(r)
```
- Выводим символ `r` на экран.
- `PrintRune` принимает `rune`; цикл `for range` дал нам тип, который идеально подходит.

---

## Заключение

Что важно усвоить из этого задания:
- как перебирать строку по символам через `for range`;
- как правильно работать с Unicode-символами (`rune`);
- как выводить данные через `z01.PrintRune`;
- что в Go строки неизменяемые, и для вывода по символам не нужна их модификация.
Это фундамент, который пригодится в последующих заданиях, особенно во всех функциях, где требуется вывести строку без использования пакета `fmt`.

# Quest 3. pointone

## Название файла Go
`pointone.go`

---

## Условия задания 

| **Instruction** | **Перевод** |
|---------------------------|--------------|
| *Write a function that takes a pointer to an int as argument and gives to this int the value of 1.* | *Напиши функцию, которая принимает указатель на int и присваивает этому числу значение 1.* |

---

## Решение

```go
package piscinego

func PointOne(n *int) {
	*n = 1
}
```

---

## Ключевые понятия

| **Термин / инструмент** | **Объяснение и назначение** |
|-------------------------------|------------------------------|
| **package piscinego** | Файл относится к пакету piscinego |
| **func PointOne(n *int)** | Функция принимает указатель на int |
| **n *int** | Тип данных: указатель на целое число |
| **n** | Хранит адрес переменной, а не значение |
| ***n = 1** | Оператор * разыменовывает указатель → присваиваем значение 1 по адресу |

---

## Что такое указатели/поинтеры (Pointers)?

- **Переменная** хранит **значение**  
- **Указатель** хранит **адрес**, где лежит значение  

Пример:

```go
x := 10
p := &x // p хранит адрес переменной x
```

### 1. Разыменование (`*p`) даёт доступ к значению по адресу

```go
*p = 99 // теперь x == 99
```

## Логика и пошаговый разбор кода

### 1. В main есть переменная:

```go
n := 0
```
n хранит значение 0.

### 2. Мы вызываем:
```go
PointOne(&n)
```
&n — это адрес переменной n.

### 3. Функция получает адрес:
```go
func PointOne(n *int)
```
То есть внутри функции n — это не число, а стрелка на число.

### 4. Мы пишем:
```go
*n = 1
```
*n = разыменовать → добраться до настоящего значения. И заменить его на 1.

## Глоссарий

| **Термин** | **Понятное объяснение** |
|-------------|--------------------------|
| **Указатель (Pointer)** | Адрес переменной в памяти |
| **&n** | Оператор «взять адрес переменной n» |
| ***n** | Разыменование: доступ к значению по адресу |
| **n *int** | Тип: указатель на целое число |
| ***n = 1** | Записываем новое значение в оригинальную переменную |

## Заключение
1. Функция принимает указатель, а не само значение.
2. Через разыменование *n изменяет значение переменной.
3. В результате переменная в main меняется, даже если мы передали её по адресу.

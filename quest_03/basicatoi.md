Нужно написать функцию, которая **принимает строку из цифр и возвращает соответствующее число в `int`**.

Строка гарантированно корректная и содержит только цифры, можно не проверять ошибки.

### Пример использования

```go
package main

import (
	"fmt"
	"piscine"
)

func main() {
	fmt.Println(piscine.BasicAtoi("12345"))     // 12345
	fmt.Println(piscine.BasicAtoi("0000123"))   // 123
}
```

---

## Решение

```go
package piscine

func BasicAtoi(s string) int {
	result := 0

	for _, ch := range s {
		// ch - '0' преобразует руну цифры в её числовое значение
		digit := int(ch - '0')

		// накапливаем число по формуле:
		// "сдвинуть число на разряд + добавить текущую цифру"
		result = result*10 + digit
	}

	return result
}
```

---

## Ключевые понятия

| **Термин / Инструмент** | **Объяснение и назначение** |
|-------------------------------|------------------------------|
| **Тип `rune`** | Каждая итерация `for range` по строке возвращает руну (Unicode-код символа). |
| **Операция `ch - '0'`** | Превращает символ `'0'..'9'` в его числовое значение. Например `'7' - '0' = 7`. |
| **Формула накопления числа** | `result = result*10 + digit` — стандартный способ собрать число из последовательности цифр. |

## Как строка превращается в число

### 1. Цифры это тоже символы

- `'0'` имеет ASCII код `48`
- `'1'` — `49`
- `'2'` — `50`
- …и так далее

Поэтому `'6' - '0' = 54 - 48 = 6`.

### 2. Как собрать число вручную

Когда вы видите строку `"123"`:

- Начинаем с `0`
- Берём `1`:  
  `result = 0*10 + 1 = 1`
- Берём `2`:  
  `result = 1*10 + 2 = 12`
- Берём `3`:  
  `result = 12*10 + 3 = 123`

Этот процесс реализуется в цикле.

### 2. Цикл по символам строки

```go
    for _, ch := range s {
```

Цикл `range` возвращает **руну**, а не байт. Открываем посимвольный перебор строки s.

---

### 3. Превращение символа в цифру

```go
        digit := int(ch - '0')
```

Берем каждую руну `ch` из строки s, отнимаем `0` (значения этих символов), превращаем в `int` и сохраняем в новой **локальной** переменной. 

---

### 4. Накопление числа

```go
        result = result*10 + digit
    }
```

Методом ручного построения числа из последовательности цифр обновляем `result` сохраняя туда нынешнее число.

---

### 5. Возврат результата

```go
        return result
```

Как только цикл проходит по всем числам строки и закрывается, возвращаем результат.

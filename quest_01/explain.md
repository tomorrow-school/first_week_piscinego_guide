# Quest 1. explain.sh

## Название файла Go
`explain.sh`

---

## Условия задания 

| **Instruction (оригинал)**                                                  | **Перевод**                                                                                |
| --------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------ |
| *"So you want to play?"*                                                    | *«Так ты хочешь сыграть?»*                                                                 |
| *"How did you do it?"*                                                      | *«Как ты это сделала?»*                                                                    |
| The commissioner thanks you for delivering the suspect.                     | Комиссар благодарит тебя за задержание подозреваемого.                                     |
| He asks how you found him.                                                  | Он спрашивает, как ты вышла на след.                                                       |
| Create **explain.sh** that:                                                 | Нужно создать **explain.sh**, который делает следующее:                                    |
| 1. displays the first and last name of your key witness                     | 1. Выводит имя и фамилию ключевого свидетеля                                               |
| 2. displays the interview number of this witness                            | 2. Выводит номер интервью этого свидетеля                                                  |
| 3. displays the colour and make of the suspect’s car                        | 3. Выводит цвет и марку машины главного подозреваемого                                     |
| 4. displays the 3 other suspects not arrested (alphabetically by last name) | 4. Выводит имена 3 других подозреваемых, не арестованных (в алфавитном порядке по фамилии) |


---

## Решение

```go
echo "Annabel Church"
echo "699607"
echo "Blue Honda"
echo "Joe Germuska"
echo "Hellen Maher"
echo "Erika Owens"
```

---

## Ключевые понятия

| Элемент                             | Объяснение                                                  |
| ----------------------------------- | ----------------------------------------------------------- |
| **echo**                            | Команда вывода текста в терминал.                           |
| `"Annabel Church"`                  | Имя и фамилия ключевого свидетеля.                          |
| `"699607"`                          | Номер интервью, связанный со свидетелем.                    |
| `"Blue Honda"`                      | Цвет и марка машины главного подозреваемого.                |
| Подозреваемые в алфавите по фамилии | Germuska, Maher, Owens — фамилии идут в правильном порядке. |


---


## Глоссарий

| Термин                         | Пояснение                                        |
| ------------------------------ | ------------------------------------------------ |
| **Shell script**               | Скрипт, написанный для bash или другой оболочки. |
| **echo**                       | Команда, которая печатает строку в терминал.     |
| **Алфавитный порядок фамилий** | Сравнение слов начиная с первой буквы фамилии.   |
| **Свидетель (witness)**        | Человек, давший важную информацию.               |


---

## Заключение

1. explain.sh выводит:
- ключевого свидетеля
- номер его интервью
- машину главного подозреваемого
- трёх не арестованных подозреваемых
2. Всё появляется в правильном порядке, как требует задание.
3. Данные подобраны из расследования mystery.

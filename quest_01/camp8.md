# Quest 1. skip.sh

## Название файла Go
`skip.sh`

---

## Условия задания 

| **Instruction (оригинал)**                                                                                                        | **Перевод**                                                                                                              |
| --------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------ |
| *"pick your equipment"*                                                                                                           | *«Выбери своё оборудование»*                                                                                             |
| *Write a command line in a skip.sh file that prints the result of a ls -l skipping 1 line out of 2, starting with the first one.* | *Напиши команду в файле `skip.sh`, которая выводит результат `ls -l`, пропуская каждую вторую строку, начиная с первой.* |

То есть нужно вывести список файлов, но только строки №2, №4, №6… (чётные строки).
---

## Решение

```go
ls -l | awk 'NR % 2 == 0'
```
---

## Ключевые понятия

| Элемент       | Объяснение                                                        |                                              
| ------------- | ----------------------------------------------------------------- | 
| `ls -l`       | Показывает файлы в длинном формате (разрешения, даты, владельцы). |                                           
| `             | ` (pipe)  Передаёт вывод предыдущей команды в следующую.          |
| `awk`         | Утилита для обработки текста построчно.                           |                                            
| `NR`          | Номер текущей строки (Number of Record).                          | 
| `NR % 2 == 0` | Выбор только чётных строк (2, 4, 6…).                             | 

---

## Логика и пошаговый разбор кода

### 1.  Передаёт вывод предыдущей команды в следующую.

```go
ls -l
```

### 2. Передаём этот вывод в awk
```go
ls -l | awk ...
```
Теперь каждая строка будет обработана awk как отдельная запись.

### 3. Фильтруем строки по номеру
```go
NR % 2 == 0
```
NR — номер строки:
1 → нечётная → пропускаем
2 → чётная → выводим
3 → нечётная → пропускаем
4 → чётная → выводим

### 4. Итог
Выводятся только строки с чётными номерами — т.е. «пропускаем одну из двух, начиная с первой».
